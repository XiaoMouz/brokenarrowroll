## 贡献指南

我们欢迎对项目的贡献！如果您想要改动，请遵循以下原则：

### 词条修改原则

1. **尊重贡献者**：不要删除某个贡献过的作者创建的内容
2. **保留多样性**：不要大量删除某个作者的词条，保持内容的多样性
3. **优化优先**：在考虑删除词条之前，尽可能尝试优化而非删除
4. **清晰备注**：进行大量变动时，请在提交或PR中备注更新的具体内容

### 贡献流程

方式1. Fork 本仓库 并修改 pull
方式2. 直接朝我要合作权限，并遵守词条修改原则
